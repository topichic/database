# Docker - Краткое руководство

Docker — это платформа для контейнеризации приложений. Она позволяет упаковать приложение со всеми зависимостями (библиотеки, настройки, код) в стандартизированный блок, называемый контейнером.

## 🎯 Основные преимущества
- **Переносимость**: Работает одинаково на любой системе с Docker
- **Изоляция**: Приложения не конфликтуют друг с другом
- **Эффективность**: Использует меньше ресурсов, чем виртуальные машины
- **Скорость**: Контейнеры запускаются за секунды

## 🚀 Быстрый старт
```bash
# Установка на Ubuntu
sudo apt update
sudo apt install docker.io

# Проверка установки
docker --version

# Тестовый запуск
docker run hello-world

# Список процессов
 docker ps -a

# Завершить все процессы
docker container prune

# Список картинок
docker images
